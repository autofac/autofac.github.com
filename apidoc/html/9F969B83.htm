Usage Note

In Visual Basic and C#, you can call this method as an instance method on any object of type IContainer. When you use instance method syntax to call this method, omit the first parameter. For more information, see Extension Methods (Visual Basic) or Extension Methods (C# Programming Guide).
Remarks

This is a convenience method that attaches the tracer to the DiagnosticListener associated with the container. If you have an event listener that isn't a DiagnosticTracerBase you can use standard DiagnosticListener semantics to subscribe to the events with your custom listener.
