Remarks

This Module is primarily used during application startup (in Global.asax) to register mappings from commonly referenced contextual application properties to their corresponding abstraction.

The following mappings are made:

Common ConstructAbstraction
HttpContext.CurrentHttpContextBase
HttpContext.Current.ApplicationHttpApplicationStateBase
HttpContext.Current.RequestHttpRequestBase
HttpContext.Current.Request.BrowserHttpBrowserCapabilitiesBase
HttpContext.Current.Request.FilesHttpFileCollectionBase
HttpContext.Current.Request.RequestContextRequestContext
HttpContext.Current.ResponseHttpResponseBase
HttpContext.Current.Response.CacheHttpCachePolicyBase
HttpContext.Current.ServerHttpServerUtilityBase
HttpContext.Current.SessionHttpSessionStateBase
HostingEnvironment.VirtualPathProviderVirtualPathProvider

In addition, the UrlHelper type is registered for construction based on the current RequestContext.

The lifetime for each of these items is one web request.
