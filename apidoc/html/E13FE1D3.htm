Remarks

Apply this extension to component registrations that use attributes that derive from the ParameterFilterAttribute like the MetadataFilterAttribute in their constructors. Doing so will allow the attribute-based filtering to occur. See MetadataFilterAttribute for an example on how to use the filter and attribute together.
